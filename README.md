An (almost) nude boilerplate with Next App router, SCSS Module, GSAP Lenis and some utils.


### How to install

Just clone it then and run <code>pnpm dev</code>!

### Packages included
- `@cher-ami/utils` 1.4.1
- `@gsap/react` 2.1.2
- `change-case` 5.4.4
- `cross-env` 7.0.3
- `graphql` 16.8.1
- `gsap` 3.13.0
- `inquirer` 12.6.3
- `lenis` 1.3.17
- `next` 16.1.6
- `react` 19.2.4
- `react-dom` 19.2.4
- `sharp` 0.32.6

### Useful commands
- `pnpm scaffold` to create some components faaaast
